# Art & Style Reference

Reference images for brutalist architecture, wall texture, window design, and damage/ destruction.

## 1. Reinforced concrete pillar damage

- **File:** `reinforced-concrete-pillar-damage.png`
- **Use for:** Reinforced concrete destruction model — concrete crumbles/shatters, rebar exposed (rust-colored rods, bent or protruding).
- **Details:** Weathered grey-beige exterior; rough darker interior where broken; thick cylindrical rebar (rust reddish-brown), bent/wavy or sticking up from broken surfaces. Damage types: deep gouge/side impact, sheared top with rebar cluster, severe stump with mangled rebar, surface cavity exposing core rebar. Sharp, jagged transition between intact face and shattered interior.

## 2. Brutalist facade (modular blocks + window framing)

- **Use for:** Overall brutalist style, wall texture, concrete crumbling effect, window cutout framing.
- **Details:** Grey cast-concrete, modular protruding/recessed blocks, vertical ribs, deep window recesses framed by thick concrete beams. Simple rectangular windows in a grid.
- **Save as:** `brutalist-facade-modular.png` (or `.jpg`)

## 3. Window in wall (stone frame)

- **File:** `window-in-wall-stone-frame.png`
- **Use for:** Building section + gribbly: a base structural wall with a window as a detail piece. Reference for how a window sits in a wall.
- **Details:** Two-toned wall — lower third rough stone blocks (sandy beige, porous, deep crevices; heavy/structural, crumble-style); upper two-thirds smooth plaster/concrete (light grey-beige, horizontal cracks, pockmarks, weathering). Window: small square opening with **thick off-white stone frame** (lintel, sill, two jambs), slightly recessed with a projecting lip; interior is a **solid matte yellow panel** (shutter/board, not glass). Strong side light casts vertical shadow for depth. Clear horizontal material transition; window breaks up the flat upper wall as an isolated gribbly.
- **In-game:** Base section = two materials (stone base + plaster upper) or single wall with segment roles; window = gribbly or opening with frame prop. Frame as chunky stone blocks; interior can be opaque (shutter) or glass.

## 4. Standard windows (ribbed concrete + square windows)

- **Use for:** Standard window design, ribbed/fluted concrete material.
- **Details:** Raw grey concrete with narrow vertical ridges and grooves (corduroy/raked). Two identical square windows, recessed, thin light-grey metal frames, single large panes, reflective glass.
- **Save as:** `brutalist-ribbed-concrete-windows.png` (or `.jpg`)

## In-game application

- **Walls:** Aim for ribbed or modular block concrete; damage = crumble (chunks + dust). Reinforced concrete = crumble + exposed rebar skeleton (see `reinforced-concrete-pillar-damage.png`). Two-toned walls (stone base + plaster upper) as in `window-in-wall-stone-frame.png` are valid section variants.
- **Windows:** Square or rectangular, recessed; frame can be minimal metal or **thick stone** (lintel, sill, jambs). Interior can be single pane glass or opaque (shutter/board). Wall frames the cutout (sill, lintel, jambs). Use as gribblies on base structural sections for variety.
